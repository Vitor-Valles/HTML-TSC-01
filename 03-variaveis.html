Variáveis - Lógica com JS

Variáveis armazenam dados temporários.


Declaração de variáveis

Existem 3 maneiras de declarar uma variável: var, let e const.

VAr tem o escopo global ou de função e pode ser atualizado frequentemente e até antes de sua declaração, var é um modo de declaração tradicional.

LET e CONST tem o escopo de bloco local.

As variáveis com var são inicializadas em underfined, as variáveis com let e const não são inicializadas.

Nomes das variáveis

O nome das variáveis possuem regras para sua declaração.

JavaScript é uma linguagem case sensitive
Não pode conter caractere
Não pode iniciar com números
Não pode conter espaço
Pode iniciar apenas com o caractere _underline
Não poder ser com nomes de palavras reservadas de linguagem

Palavras Reservardas do JS

Palavras chaves são inderficadores pré reservado, como por exemplo os comandos de linguagem programação.

Exemplos de palavras reservadas: abstrat, byte, int, import, boolean, default, char, do, for, goto, case, continue, delete, etc...

Variável composta ou Vetor

Vetor é um tipo de estrutura de dados, onde a variável consegue armazenar simultaneamente diferente valores de um mesmo tipo.

Assim, vetor é uma variável composta, homogênea e unidimensional.

Em programação, as estruturas de dados são uma forma coerente e racional de organizar os dados otimizar o seu uso. Um vetor unidimencional, ou array, é uma forma coerente de racional de organizar os dados e otimizar o seu uso.

Um vetor unidimencional, ou array, é uma das estruturas de dados mais comuns a serem aplicadas no desenvolvimentos de um algoritmo e pode ser cisiderado uma sequencia de posições de memória que é usadas par aarmazar um conjunto de dados do mesmo tipo e usado no mesmo nome.

Para acessar cada infromação armazenada, separadamente, usa-se um índice que é represensentado uma variável inteira. Esssas informações são chamadas de elementos do vetor.

Exemplo:

notas = [5, 8, 10, 9];

Tipos de dados JS - Data Types

null - Não valor, ou inexistênte de valor associado auma variável.

underfined - Conteúdo de variável não iniciado

boolean - verdadeiro(true) ou falso (false)

string - para uma sequencia de zeros e caracteres, normalmente repsenta textos

number - Números inteiros e flutuantes

NaN - Not a number, números inválidos

object - Para coleção de propriedades

symbol - Obsoletos usado para objetos
