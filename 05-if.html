IF Decisão Condicional

O bloco Condicional mais comum é o if, que recebe um valor booleano como perametroe reotnar veradeiro ou falso de acordo com a condição.

Sintaxe de if simples, apenas com o verdadeiro: if(){};

Sintaxe de if composta, com verdadeiro e falso: if(Condição){ação vedadeira}else{ação falso};

Exercício

Crie um algorítimo para calcular a média simples de dois bimestres escolares
