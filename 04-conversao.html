Conversão - Lógica com JS

Tipos de dados

Os tupos de dados em JavaScript são: string, number e boolean.

Number

Este tipo realiza operações aritiméticas com maior facilidade, pois todas as conversões necessárias sõa realizados automaticamente pelo interpretador/browser.

Objeto String Maiúsculo

A conversão de caias e uma string pemite-nos converter todo o seu conteúdo em letras grandes e pequenas.

Convertendo

O JS possui funções de conversão de string para number.
